# 2021.9 / JavaScript 月刊 - 敏捷已死，敏捷万岁

![](./img/09.jpeg )

敏捷宣言（Agile Manifesto，敏捷软件开发）诞生至今刚好 20 年，有两个事实似乎不言自明：

1. 作为一个标签，敏捷是胜利者；没有人希望被称为“非敏捷”。
2. 但敏捷在实践上和创始人的革命性思想相去甚远。
