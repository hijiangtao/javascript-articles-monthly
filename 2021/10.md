# 2021.10 / JavaScript 月刊 - JS Logo 发布十年啦

![](./img/10.jpeg )

十年前在 JSConf EU 2011 上，用 “JS” 字母 代表 JavaScript 的社区 logo 正式发布。起初，其被用于标注 JS 项目，在 MIT 协议下，如果你愿意你也可以随意的使用它，而不用担心相关版权、税与授权许可等麻烦。详情见 <https://github.com/voodootikigod/logo.js>
